# Fitness Tracker - GitHub Pages Ready

This package is ready to upload directly to a GitHub repository and publish with GitHub Pages.

## Publish steps

1. Unzip this folder.
2. Upload all files in this folder to the root of your GitHub repository.
3. In GitHub, open **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** main
   - **Folder:** / (root)
5. Save.
6. Wait for GitHub Pages to publish, then open your site URL.

## Files

- `index.html` - app entry page
- `app.jsx` - your attached React component adapted for browser use
- `.nojekyll` - avoids Jekyll processing on GitHub Pages
